# Practice_1_HP_Aruzhan_Bissimbayeva
Динамическая память и указатели на С++

## Описание задания

В третьей части практической работы реализована программа на языке C++, демонстрирующая
работу с динамической памятью и указателями, а также основы параллельных вычислений
с использованием технологии OpenMP.

Цель задания - научиться:
- создавать динамический массив с помощью указателей;
- работать с динамически выделенной памятью;
- реализовывать вычисления в отдельной функции;
- использовать директиву OpenMP `reduction` для параллельного подсчёта;
- корректно освобождать память после завершения работы программы.

---

## Что делает программа

1. Пользователь вводит размер массива `N` с клавиатуры.
2. В программе динамически выделяется массив целых чисел размером `N` с помощью оператора `new`.
3. Массив заполняется случайными числами в диапазоне от 1 до 1000.
4. Если размер массива небольшой, его содержимое выводится на экран.
5. Для вычисления среднего значения элементов массива вызывается отдельная функция.
6. Внутри функции выполняется параллельный подсчёт суммы элементов массива
   с использованием директивы OpenMP:
   ```cpp
   #pragma omp parallel for reduction(+:sum)
7. После вычисления среднего значения результат выводится в консоль.
8. В конце программы динамически выделенная память освобождается с помощью delete[].

## Используемые технологии

-Язык программирования: C++
-Компилятор: g++
-Параллельное программирование: OpenMP
-Динамическая память и указатели

## Компиляция (с поддержкой OpenMP)

В терминале выполните команду:
 ```cpp
   g++ lab3.cpp -fopenmp -o lab3.exe
   .\lab3.exe 
 ```
  После запуска программа запросит размер массива и выведет результат вычислений.
  
## Примечания

- При больших размерах массива вывод его содержимого в консоль отключён,
чтобы избежать перегрузки вывода.

- Среднее значение отличается при каждом запуске программы,
так как массив заполняется случайными числами.

- Использование reduction гарантирует корректность вычислений
при параллельной обработке данных.
